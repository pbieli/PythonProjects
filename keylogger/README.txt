Keylogger Script

Developed a keylogging application in Python to capture keystrokes from the user and log them to a file for analysis.

-Utilized the pynput library for monitoring keyboard events in real-time.
-Implemented a callback function to handle key press events and write them to a log file.
-Incorporated a stop flag mechanism to control when to stop the keylogging process.
-Enhanced user experience by providing a clear message indicating how to stop the keylogger.
-Demonstrated proficiency in Python programming and system-level interaction.

Technologies Used: Python, pynput library, system-level interaction
